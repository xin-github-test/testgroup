1.dir
 查看当前目录
2.切换磁盘 cd /d
 cd /d f:    切换到f盘，也可以直接使用 f:s 
3.切换目录 cd 
4.回到上一级目录 cd ..
5.回到根目录 cd \
6.创建目录 ：md 名称
 新建多个文件夹 md 名称1 名称2...
7.删除空目录
 rd 名称
8.删除多层次的目录：
 rd /q/s 名称     /q :不出提示  /s:多层次删除
 不带参数 /q 删除前会让你确认
9.向文件中写入数据
 echo 内容 > d:\test\abc\abc.txt    向d盘下的test下的abc 下的abc.txt中写入指定的内容
 也可以以 使用 echo 内容 > 文件名  在本目录新建一个名称为该文件名的文件，并将指定的内容写入
10.文件的拷贝
 copy 文件名 绝对路径  将文件拷贝到相应的目录下，也可以在绝对路径下加上新的文件名，使得文件移动后，文件名也相应改变
11.剪切文件
 move 文件名 绝对路径 （文件名） 将目录下的文件移动到指定的目录，也可以修改移动后的文件名
12.删除文件
 del 文件名 ，文件名也可以替换为绝对路径 ，使得删除其它目录的文件
13.使用通配符删除多个文件
 del *.txt  删除以.txt结尾的文件
14.清屏
 cls
15.退出 
exit
16.ren 重命名 
ren 旧名 新名
17.type 查看文件中的内容
18.对文件进行加权限处理
 attrib 文件 +参数    H：为隐藏
 删除权限  用 - 就可以了
19.查看端口等信息
netstat -na
20.关闭计算机
 shutdown    -r 重启  -t 时间， 设置多久关机或重启
21.任务管理器
tasklist  列出后台任务
taskkill  /pid  根据pid 删除后台任务
22.net user 用户名 密码，可以设置密码
23.磁盘管理
defrag 磁盘名 加上-f可以对碎片进行整理
24.net user 名字 密码 /add 添加用户
   net localgroup 组名  查看改组中的成员
 net localgroup 组名 用户名 /add 将成员加入某个组
